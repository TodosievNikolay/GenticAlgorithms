This project is a Python implementation of a Genetic Algorithm

## What I practiced in this project

- Python basics (functions, lists, loops)
- Reading and parsing `.tsp` files
- Euclidean distance calculation
- Greedy algorithm
- Genetic Algorithm:
  - tournament selection
  - ordered crossover
  - inversion mutation
  - elitism
- Comparing algorithms using console output
